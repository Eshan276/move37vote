Create a User

curl -X POST http://localhost:4000/users \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Alice",
    "email": "alice@example.com",
    "passwordHash": "hashedpassword"
  }'

Create a Poll

curl -X POST http://localhost:4000/polls \
  -H "Content-Type: application/json" \
  -d '{
    "question": "What is your favorite color?",
    "options": ["Red", "Blue", "Green"],
    "creatorId": 1
  }'


Get All Polls
curl http://localhost:4000/polls


Get a Specific Poll
curl http://localhost:4000/polls/1


Cast a Vote
curl -X POST http://localhost:4000/votes \
  -H "Content-Type: application/json" \
  -d '{
    "userId": 1,
    "pollId": 1,
    "optionId": 2
  }'
